# CodeForge Frontend

This is the Next.js frontend directory for the CodeForge platform.

## Getting Started

1. `npm install`
2. `npm run dev`

The dev server will open on `localhost:3000`.

**Note:** Ensure the backend judge service is running on port 7000 (check the `../judge/` folder) before you try to fetch problems. If the port is different, just update `API_BASE` inside `src/lib/api.ts`.
